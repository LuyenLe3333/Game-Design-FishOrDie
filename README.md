## FishOrDie (Unity 2D Game Development)

A 2D Unity game developed in a 3-person team using C# and Git version control.

### Key Features
- Built core gameplay systems in Unity using C#
- Designed an event-driven audio system for gameplay interactions (reeling, water, impact, UI)
- Implemented state-based background music transitions for menu, gameplay, and game-over states
- Collaborated using Git for version control and team workflow
